# COVID-19-Data-Analysis-Project

This project provides an in-depth analysis of COVID-19 data to understand the pandemic's impact across countries and continents. Using real-time datasets, it highlights trends in case counts, deaths, recoveries, and vaccination coverage. The insights from this analysis can support public health strategies and decision-making.

Skills & Tools Used: Data Analysis & Visualization: Power BI
Data Sources: COVID-19 datasets (cases, deaths, recoveries)
Techniques: Trend analysis, comparative visualization, KPI tracking, map-based insights

Objectives:
1) Analyze global COVID-19 case, death, and recovery trends.
2) Compare COVID-19 impact across countries and continents.
3) Evaluate the effect of population on COVID-19 spread.
4) Forecast potential trends for better preparedness.

Key Insights:
1) Total Confirmed Cases: 1.07 billion
2) Total Deaths: 10.42 million
3) Total Recovered: 1 billion
4) Global Death Rate: 0.97%
5) Recovery Rate: 96%
6) COVID-19 spread varied significantly across continents, with Europe, Asia, and North America being most affected.
7) Data visualization highlights the top countries by cases and deaths, along with regional trends over time.
